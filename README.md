# Torre De Hanoi - Simulação em Python

Projeto da disciplina Algoritmos e Estruturas de Dados do IFPE Campus Igarassu, demonstrando o clássico problema da Torre de Hanói implementado em Python com recursividade e interface gráfica usando Turtle.

# O que é a Torre de Hanói?

Quebra-cabeça matemático que consiste em três hastes e um número de discos de tamanhos diferentes. O objetivo é mover toda a pilha de discos da primeira para a última haste, seguindo duas regras:

✅ Apenas um disco pode ser movido por vez

✅ Um disco maior nunca pode ficar sobre um menor

# Funcionalidades

Interface gráfica interativa

Seleção de discos (1 a 8)

Animação suave dos movimentos

Contador de movimentos em tempo real

Visualização da recursividade em ação

# Conceitos Demonstrados

✅ Recursividade - Algoritmo divide e conquista

✅ Pilha de execução - Gerenciamento de chamadas recursivas

✅ Complexidade algorítmica - O(2ⁿ) movimentos

✅ Orientação a objetos - Estrutura do código
